Python Flask Developing an AI-Based Plant Predictor
Here is a complete AI-Based Plant Predictor web application using Python Flask, Machine Learning (Random Forest), and a clean user interface.
Features:
- Predicts the best plants to grow based on soil and environmental conditions.
- Uses NPK (Nitrogen, Phosphorus, Potassium), temperature, humidity, pH, and rainfall.
- Trained using a real plant recommendation dataset.
- Beautiful HTML frontend with Bootstrap.
